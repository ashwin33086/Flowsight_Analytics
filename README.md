# Flowsight_Analytics Overview
FlowSight is an advanced predictive analytics dashboard designed to transform project management from tracking to decision-making. It integrates sprint data from tools like Jira, GitHub, and Slack, applies machine-learning models, and provides real-time insights that help teams predict delays, optimize workload, and improve delivery reliability.
# Key Features
Predictive Sprint Risk Scoring
Velocity & Burndown Forecasting
Team Workload & Health Insights
Blocker Analysis & Delay Prediction
Smart Alerts & Recommendations
Continuous Learning Model
